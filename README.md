## ✨ 这是什么？

**蜂巢文件加密（LockHive）** 是一款 Telegram 文件中转 / 分享工具。  
把图片、视频、文档发出去 → 填写描述 → 立刻拿到 **密钥 `LH_xxxx`** 和 **深度链接**，发给任何人即可取回整包文件。

适合：资源合集分享、频道备份、多人协作传文件、临时云盘式分发。

---

## 🚀 快速开始

1. 发送 `/start` 进入主界面  
2. 点 **📥 存储**（或 `/new`）上传文件  
3. 输入描述（≥ 5 个字符）  
4. 获得密钥与分享链接  
5. 对方点 **📤 获取**（或 `/get`）输入密钥，或直接点链接取文件  

> 💡 也可直接把密钥 / `LH_` 码发到对话里，系统会自动识别。

## 🧩 核心功能

| 功能 | 说明 |
|------|------|
| 📥 **一键存储** | 支持多文件连续上传，确认后生成唯一密钥 |
| 📤 **密钥获取** | 输入密钥或点击深度链接即可下载 |
| 👤 **我的文件** | 查看自己上传过的文件、密钥与分享链接 |

| 🌐 **多语言** | 内置 26 种语言，`/lang` 随时切换 |
| 🔍 **密钥搜索生态** | 可使用蜂巢密钥搜索器检索 |
| 🛡 **风险拦截** | 非法 / 风险文件自动标记与拦截，符合平台规则 |
